# wan_g 的个人博客

基于 [Hexo](https://hexo.io/) 搭建的个人博客，使用 [Butterfly](https://github.com/jerryc573/hexo-theme-butterfly) 主题。

## 访问地址

[wange.me](https://wange.me)

## 技术栈

- **框架**: Hexo 8.1.1
- **主题**: Butterfly
- **部署**: GitHub Pages
- **语言**: 中文

## 博客内容

- 技术分享（Git、Node.js、npm 等开发工具）
- 学习笔记（计算机基础知识）

## 本地开发

```bash
# 安装依赖
npm install

# 启动本地服务
npm run server

# 构建静态文件
npm run build

# 部署
npm run deploy
```

## 文章结构

```
source/
└── _posts/     # 文章存放目录
```
