# 大计 - 计算机基础笔记

## 1.1 计算机发展阶段、特点、分类

### 计算机定义

按照事先储存的程序，自动、高速、精确地输入、处理、输出、存储数据的设备。

### 重要人物与里程碑

- **人工智能之父**：英国 艾伦·图灵，首先提出电子计算机的理论模型
- **第一台通用计算机**：ENIAC (1946) 美国宾夕法尼亚大学，按十进制运算
- **第一台冯·诺依曼结构计算机**：EDVAC，采用二进制

### 计算机发展阶段

|  代数  |   时间   | 元器件              | 特点                     |
| :----: | :-------: | :------------------ | :----------------------- |
| 第一代 | 1946-1958 | 电子管              | 体积大、功耗高、可靠性低 |
| 第二代 | 1958-1964 | 晶体管              | 体积缩小、性能提升       |
| 第三代 | 1964-1971 | 中小规模集成电路    | 集成度提高               |
| 第四代 | 1971至今 | 大/超大规模集成电路 | 微型化、高性能           |

### 计算机分类

**按处理方式**：

- 数字计算机（0101）
- 模拟计算机（电信号）
- 数字模拟混合计算机

**按用途**：

- 专用计算机
- 通用计算机

### 计算机特点

1. 高速精确运算
2. 准确逻辑判断
3. 强大存储能力
4. 自动功能
5. 网络与通信能力

### 主要应用领域

- **科学计算**：复杂数学运算
- **数据和信息处理**：办公自动化、数据库
- **过程处理**：工业控制、实时监控
- **计算机辅助**：
  - CAD - 计算机辅助设计
  - CAI - 计算机辅助教学
  - CAM - 计算机辅助制造
  - CAT - 计算机辅助测试
- **人工智能**：机器学习、专家系统
- **多媒体应用**：图像、音频、视频处理
- **嵌入式系统**：智能设备、物联网

### 发展趋势

- 巨型化：超级计算机
- 微型化：便携设备
- 网络化：互联网、云计算
- 智能化：AI 应用

## 1.2系统组成

### 冯·诺依曼体系结构

五大基本部件：

- **运算器**：执行算术和逻辑运算
- **存储器**：存储程序和数据
- **控制器**：控制各部件协调工作
- **输入设备**：将外部信息送入计算机
- **输出设备**：将结果送出计算机

### 计算机系统组成

```
计算机系统
├── 硬件
│   ├── 主机
│   │   ├── CPU（中央处理器）
│   │   │   ├── 运算器
│   │   │   └── 控制器
│   │   └── 内存（主存）
│   └── 外部设备
│       ├── 外部存储器（外存）
│       ├── 输入设备（Input Device）
│       └── 输出设备（Output Device）
└── 软件
    ├── 系统软件
    │   └── 操作系统
    └── 应用软件
        ├── 字处理软件
        ├── CAD
        └── CAI 等
```

### 软件系统

**系统软件**：

- 操作系统：UNIX / Linux / Windows
- 语言处理程序：汇编程序，解释程序，编译程序
- 程序设计语言：机器语言（0101），汇编语言（比机器语言多了助记符），高级语言（c++)

**应用软件**：解决各类具体问题（ps、Word 等）

### 性能指标

| 指标                       | 说明                                                                |
| -------------------------- | ------------------------------------------------------------------- |
| **字长**             | CPU 一次能处理的二进制位数，常见有 8/16/32/64 位                    |
| **运算速度**         | 每秒执行指令条数，单位：MIPS（百万条指令/秒）                       |
| **主频**             | 时钟脉冲信号频率，单位 MHz / GHz                                    |
| **内存容量**         | 基本单位 Byte（B）                                                  |
| **存储容量换算**     | 8 bit = 1 B，1 K = 1024 B，1 M = 1024 K，1 G = 1024 M，1 T = 1024 G |
| **内存储器工作频率** | 内存最高工作频率，单位 MT/s                                         |
| **总线带宽**         | 总线每秒能传输的数据量 = (总线频率 × 数据宽度) / 8                 |
| **核心数**           | 计算核心数，有单核 / 双核 / 4 核 / 8 核 / 12 核心等                 |
| **网络带宽**         | 数据传输速率单位 bit/s，有 Kbits、Kbits/s、Mbits/s 等               |

> 度量 CPU 性能时，主时钟频率单位为 GHz。
